# DarshanFx Portfolio Website

A static portfolio website for DarshanFx, the graphic design portfolio of Kanhaiyalal Suthar from Rajasthan, India.

## Files

- `index.html` - Main website markup, SEO metadata, schema data, and page sections.
- `style.css` - Responsive styling and layout.
- `script.js` - Mobile menu, portfolio filtering, scroll reveal, back-to-top button, year update, and contact form mail fallback.
- `robots.txt` - Search crawler rules.
- `sitemap.xml` - Basic sitemap for the homepage.
- `assets/` - SVG placeholder portfolio previews.

## Local Development

Open `index.html` directly in a browser, or serve the folder with any static server.

Example:

```bash
python -m http.server 8080
```

Then open:

```text
http://127.0.0.1:8080/
```

## Customize Before Publishing

- Replace `https://yourusername.github.io/darshanfx/` with the real website URL.
- Replace the placeholder WhatsApp number `910000000000`.
- Replace `hello@darshanfx.com` with the real email address.
- Replace social profile placeholder links in the footer.
- Replace SVG placeholder portfolio previews in `assets/` with real project images when available.
